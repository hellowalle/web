1. 360度全景展示
     
     示列地址: http://www.arhome.cn/vtour195/

     优点: 身临其境的体验方式

     应用场景: 汽车内饰360演示, 房屋装修360演示, 虚拟显示互动游戏等等.

     实现方法: 拍摄全景照片, 使用KRPano提供的工具生成web版本, 支持flash与html5

    研究方向: html5 CSS 3D Transforms, WebGL
